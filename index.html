SplitCsv

Command-line utility to split large CSV files into multiple smaller ones, while keeping header rows intact. Author: Nate Reed (@natereed).

Usage:

splitcsv.py [-h] [--output_header_row] [--input_header_row] path size

Arguments

  • -h for help
  • --output_header_row: If a header row is provided in input, output this row in each file.
  • --input_header_row: Treat the first row of input as a header row.
